# BEP1-Agent-System
本项目是一个前后端分离的代理系统。

## 前端运行说明
本项目的前端为静态HTML页面，无需安装复杂环境，运行方式如下：
1.  进入项目的 `frontend` 文件夹，找到 `index.html` 文件。
2.  双击该文件，用任意浏览器（Chrome、Edge、Firefox等）打开，即可访问前端页面。
3.  如需和后端联调，请先启动后端服务（参考 `fastapi-project/README.md` 中的后端启动说明），再刷新前端页面即可。

## 后端运行说明
后端服务使用FastAPI构建，具体配置与启动方式请参考 `fastapi-project/README.md` 文件。
